Use this with Claude to get personalized resource recommendations based on Xiaoyang Hu's curation system.

## Overview

A curated collection of design resources for designers and web coders. Each resource is here because it earned its place — filtered through real usage, with personal notes on why it matters.

## Categories

### 1. X Accounts to Follow
Designers worth following on X for daily aesthetic intake. Work at Vercel, Figma, Linear, OpenAI. Tier 1 must-follows: sam, haris, Ryan Stephen, Ryo Lu, rauno, Jakub Krehel, Ilya, Ridd, Zsolt Kálmán, Oğuz Yağız Kara, Quartr, Elena.

### 2. Design Inspiration
Visual reference, motion galleries, interaction inspiration. Tier 1: Design Spells. Also: App Motion, Landingfolio, awwwards, GSAP, Spline, Rauno Freiberg, Things Inc.

### 3. UI Component Libraries
Front-end components for web coding. Tier 1: React Bits, FreeFrontend. Also: Magic UI, Aceternity UI, Fancy Components, Uiverse, Neobrutalism.

### 4. Framer Resources
Framer ecosystem — components, templates, UI kits. Tier 1: Framer University. Also: Remix Lab, framer.today, Cursors, Segment UI.

### 5. Portfolio Inspiration
Curated designer portfolios. Tier 1: alexyc, Elisha Jeon, Arshad Riyas, Serena Li, Alric Monteiro, Jackie Zhang, Xiaoyang Hu.

### 6. Portfolio Platforms
Aggregators for browsing portfolios. Tier 1: Bestfolios, Open Doors. Also: Wall of Portfolios.

## Usage Context

- **Building a portfolio?** Start with Portfolio Inspiration tier 1, then X Accounts for daily inspiration.
- **Learning web coding?** UI Components (React Bits, Magic UI) + Design Inspiration (Design Spells).
- **Using Framer?** Framer University, then Remix Lab and Segment UI.
- **Need motion reference?** GSAP, App Motion, Design Spells.
